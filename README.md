# DevDNA

Agent Skills 开源工程。面向 Cursor、Claude Code、OpenAI Agent 等多平台，构建工业级 AI Skills Library。

## 架构原则

- **标准化**：统一的 Skill 规范与接口约定
- **模块化**：可复用、可组合的 Skill 单元
- **工程化**：生产就绪，交付物工业级
- **多平台**：兼容主流 AI Coding Agent

## 项目结构

```
DevDNA/
├── specs/          # Skill 输出规范
├── skills/         # Skill 实现
├── examples/       # 使用示例
└── docs/           # 工程文档
```

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/YouzSpace/DevDNA.git
cd DevDNA

# 安装依赖（如有）
# pnpm install
```
